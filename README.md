# Présentation Interactive — Jour des métiers

Présentation HTML interactive pour le Jour des métiers à l'école.
Conçue pour des enfants de 9-10 ans.

## 🚀 Lancement

**🔗 Lien direct :** https://louischarlesbouchard-stack.github.io/Semaine-des-metiers/

**Ou** double-cliquer sur `Lancer Présentation.url` dans le dossier.

> ⚠️ Ne PAS ouvrir `index.html` directement — le QR code cellulaire ne fonctionnera pas.

## Fonctionnalités
- Applaudimètre micro
- Scanner de visages IA (face-api.js)
- Vote par swipe via cellulaire (PeerJS)
- Buzzer quiz avec équipes
- Photo magique avec analyse IA Claude
- Micro baladeur avec sous-titres temps réel
- Chatbot live Claude API

## Configuration requise
- Clé API Anthropic (à entrer au démarrage via bouton dédié)
- Chrome récent
- Caméra + micro

## Touches clavier
- F : plein écran
- P : mode panique (désactive toutes fonctions interactives)
- R : reset votes
- ←/→ : navigation slides
- 1/2 : vote fallback (si cell déconnecté)
